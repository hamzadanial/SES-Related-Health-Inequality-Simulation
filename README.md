# SES-Related Health Inequality Simulation Study

This repository contains the R code and simulation data used in the paper:

"A Regression-Based Framework of Socioeconomic-Related Health Inequality"

## Contents

- Simulation data generation
- Monte Carlo simulation study
- Proposed SES-related inequality indices
- Conventional inequality measures
- Final simulation tables

## Files

### R/
- SES_related_inequality_data_generation.R
- Simulation_study_SES_related_health_inequality.R

### data/
- SES_simulation_population.csv

### results/
- Final simulation study tables

## Software

- R version 4.x
- Packages:
  - data.table
  - DescTools

## Reproducibility

Running the scripts reproduces all simulation results and tables reported in the paper.
